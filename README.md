# PURE MED - Traditional Tamil Natural Medicine & Remedies

**Discover natural healing solutions rooted in ancient Tamil Siddha medicine and traditional Paatti Vaithiyam (grandmother's remedies)**

## Overview

PURE MED is an interactive web application that brings traditional Tamil natural medicine to your fingertips. With over 140+ carefully curated remedies, comprehensive herb guides, and an intelligent symptom checker, explore how Tamilians have healed themselves naturally for generations.

Whether you're searching for relief from a common cold, exploring medicinal herbs, or discovering age-old home remedies, PURE MED connects you with natural wellness solutions grounded in centuries of Tamil healing traditions.

## Getting Started

### Installation

1. **Clone or download the repository**
   ```bash
   git clone <repository-url>
   cd PURE MED
   ```

2. **Open in browser**
   - Simply open `index.html` in your web browser
   - No server or build tools required
   - Works offline once loaded

### System Requirements
- Modern web browser (Chrome, Firefox, Safari, Edge)
- JavaScript enabled
- Recommended: 1366x768px or larger screen for optimal experience

## How to Use

### Finding a Remedy

1. **Search Method**
   - Use the search bar to find remedies by disease name (e.g., "Cough", "Migraine")
   - Or search by ingredient (e.g., "Turmeric", "Ginger")
   - Click "View Recipe" for detailed preparation instructions

2. **Browse by Category**
   - Navigate to "Explorer" tab
   - Select a category (Respiratory, Digestion, etc.)
   - Explore all remedies in that category

3. **Use Symptom Checker**
   - Click "Symptom Checker" tab
   - Follow the 3-step wizard:
     - **Step 1**: Select affected body system
     - **Step 2**: Choose specific symptoms
     - **Step 3**: Get personalized remedy suggestions

### Explore Herbs

- Visit the "Herbs Guide" tab
- Learn about medicinal properties of 50+ common herbs
- Understand traditional uses and health benefits
- Find recipes and preparation methods

### Save Favorites

- Click the bookmark icon on any remedy card
- View all saved remedies in the "Saved" tab
- Quickly access your personal wellness library

## Database Structure

### Remedies Database
Each remedy includes:
- **Disease Name** (English & Tamil)
- **Category** (Respiratory, Digestion, Pain, Skin, etc.)
- **Ingredients** (with quantities)
- **Preparation Method** (step-by-step instructions)
- **Dosage** (how much to take and frequency)
- **Benefits** (health advantages)
- **Precautions** (when to avoid)

### Herbs Encyclopedia
Each herb entry contains:
- **Herb Name** (English & Tamil)
- **Medicinal Properties**
- **Common Uses**
- **Health Benefits**
- **Preparation Methods**
- **Storage Tips**
